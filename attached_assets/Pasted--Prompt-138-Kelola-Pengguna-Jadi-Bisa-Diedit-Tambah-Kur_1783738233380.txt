## Bagian A — SQL Migrasi (Siapkan untuk Saya Jalankan Manual)

```sql
-- Cek dulu policy RLS enrollments yang sudah ada sebelum nambah baru
SELECT policyname, cmd FROM pg_policies WHERE tablename = 'enrollments';
```

Tambahkan (kalau belum ada) policy admin untuk kelola enrollment siapa saja:
```sql
CREATE POLICY "enrollments: admin bisa insert siapa saja"
  ON enrollments FOR INSERT
  WITH CHECK (
    EXISTS (SELECT 1 FROM user_profiles WHERE user_profiles.user_id = auth.uid()::text AND user_profiles.is_admin = true)
  );

CREATE POLICY "enrollments: admin bisa delete siapa saja"
  ON enrollments FOR DELETE
  USING (
    EXISTS (SELECT 1 FROM user_profiles WHERE user_profiles.user_id = auth.uid()::text AND user_profiles.is_admin = true)
  );
```
(Policy user biasa yang insert/delete enrollment MILIK SENDIRI — dari
proses checkout normal — TETAP ADA, tidak dihapus, policy baru ini
cuma nambah cakupan untuk admin.)

Tabel baru untuk grant kelas ke email yang belum pernah login (pola
sama seperti `admin_invites`):
```sql
CREATE TABLE IF NOT EXISTS class_grants (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email TEXT NOT NULL,
  class_id UUID NOT NULL REFERENCES classes(id) ON DELETE CASCADE,
  granted_by TEXT,
  granted_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  redeemed_at TIMESTAMPTZ,
  UNIQUE (email, class_id)
);

ALTER TABLE class_grants ENABLE ROW LEVEL SECURITY;

CREATE POLICY "class_grants: admin bisa kelola"
  ON class_grants FOR ALL
  USING (
    EXISTS (SELECT 1 FROM user_profiles WHERE user_profiles.user_id = auth.uid()::text AND user_profiles.is_admin = true)
  )
  WITH CHECK (
    EXISTS (SELECT 1 FROM user_profiles WHERE user_profiles.user_id = auth.uid()::text AND user_profiles.is_admin = true)
  );
```

## Bagian B — Edge Function `check-admin-invite`: Sekalian Redeem `class_grants`

Edit `supabase/functions/check-admin-invite/index.ts` (function ini
SUDAH jalan tiap kali user login, tinggal ditambah logic baru di
dalamnya, JANGAN bikin edge function terpisah supaya tidak perlu
trigger baru):

Setelah bagian existing yang cek `admin_invites`, TAMBAHKAN logic baru:
```ts
// Cek juga apakah ada class_grants untuk email ini yang belum diredeem
const { data: grants } = await supabaseAdmin
  .from('class_grants')
  .select('id, class_id')
  .eq('email', normalizedEmail)
  .is('redeemed_at', null);

if (grants && grants.length > 0) {
  const enrollments = grants.map((g: any) => ({ user_id: user.id, class_id: g.class_id }));
  await supabaseAdmin
    .from('enrollments')
    .upsert(enrollments, { onConflict: 'user_id,class_id', ignoreDuplicates: true });

  await supabaseAdmin
    .from('class_grants')
    .update({ redeemed_at: new Date().toISOString() })
    .in('id', grants.map((g: any) => g.id));
}
```
(Pastikan variable `normalizedEmail`/`user` yang dipakai konsisten
dengan yang sudah ada di function ini untuk bagian admin_invites —
reuse, jangan deklarasi ulang.)

## Bagian C — Function di `lib/db.ts`

```ts
export async function listUserEnrollments(userId: string): Promise<{ classId: string; classTitle: string }[]> {
  const { data, error } = await supabase
    .from('enrollments')
    .select('class_id, classes(title)')
    .eq('user_id', userId);
  if (error) throw error;
  return (data ?? []).map((e: any) => ({ classId: e.class_id, classTitle: e.classes?.title ?? '(kelas terhapus)' }));
}

export async function adminAddEnrollment(userId: string, classId: string): Promise<void> {
  const { error } = await supabase.from('enrollments').insert({ user_id: userId, class_id: classId });
  if (error) throw error;
}

export async function adminRemoveEnrollment(userId: string, classId: string): Promise<void> {
  const { error } = await supabase.from('enrollments').delete().eq('user_id', userId).eq('class_id', classId);
  if (error) throw error;
}

export async function createClassGrant(email: string, classIds: string[]): Promise<void> {
  const { error } = await supabase
    .from('class_grants')
    .insert(classIds.map((classId) => ({ email: email.toLowerCase().trim(), class_id: classId })));
  if (error) throw error;
}

export async function listPendingGrantsForEmail(email: string): Promise<{ classId: string; classTitle: string }[]> {
  const { data, error } = await supabase
    .from('class_grants')
    .select('class_id, classes(title)')
    .eq('email', email.toLowerCase().trim())
    .is('redeemed_at', null);
  if (error) throw error;
  return (data ?? []).map((g: any) => ({ classId: g.class_id, classTitle: g.classes?.title ?? '' }));
}
```
