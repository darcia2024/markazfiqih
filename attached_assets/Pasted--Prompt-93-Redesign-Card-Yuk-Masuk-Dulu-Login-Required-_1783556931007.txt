# Prompt 93: Redesign Card "Yuk, Masuk Dulu" (Login Required State)

## Konteks
Card yang muncul saat user belum login mencoba akses halaman terproteksi
(`ProtectedRoute.tsx`) sekarang terlalu polos — cuma ikon generik `BookLock`
dan card putih flat. Redesign supaya lebih menarik dan konsisten dengan
identitas visual Markaz Fiqih yang sudah dipakai di `LoginPage.tsx`
(gradient merah, tekstur `hero-pattern.png`, aksen gold), sekaligus
tampilkan logo asli Markaz Fiqih (bukan ikon generik).

## File yang Diubah
`artifacts/markaz-fiqh/src/components/ProtectedRoute.tsx`

Ganti seluruh isi file dengan kode berikut (logic auth TIDAK berubah,
HANYA tampilan visual saat `!user` yang di-redesign):

```tsx
import React from 'react';
import { useAuth } from '@/context/AuthContext';
import { useLocation, Link } from 'wouter';
import { Button } from '@/components/ui/button';
import { LogIn } from 'lucide-react';
import { Navbar } from '@/components/Navbar';

export function ProtectedRoute({ children }: { children: React.ReactNode }) {
  const { user, isLoading } = useAuth();
  const [location] = useLocation();

  if (isLoading) {
    return (
      <div className="min-h-screen flex flex-col bg-background">
        <Navbar />
        <div className="flex-1 flex items-center justify-center">
          <div className="h-8 w-8 animate-spin rounded-full border-4 border-primary border-t-transparent" />
        </div>
      </div>
    );
  }

  if (!user) {
    return (
      <div className="min-h-screen flex items-center justify-center bg-[hsl(var(--brand-red-tint))] p-4 relative overflow-hidden">
        {/* Tekstur pattern lembut di background, konsisten dengan LoginPage */}
        <div
          aria-hidden="true"
          className="pointer-events-none absolute inset-0 opacity-[0.04]"
          style={{
            backgroundImage: "url('/hero-pattern.png')",
            backgroundSize: '700px',
            backgroundRepeat: 'repeat',
          }}
        />

        <div className="relative bg-white rounded-[20px] shadow-xl border border-[hsl(var(--brand-gold-pale))] p-10 max-w-md w-full text-center">
          {/* Badge logo — gradient merah + tekstur pattern + ring gold tipis */}
          <div className="relative mx-auto w-20 h-20 mb-6">
            <div className="absolute inset-0 rounded-2xl bg-gradient-to-br from-primary to-[hsl(var(--brand-red-hover))] shadow-lg shadow-primary/30" />
            <div
              aria-hidden="true"
              className="absolute inset-0 rounded-2xl opacity-20 mix-blend-overlay"
              style={{
                backgroundImage: "url('/hero-pattern.png')",
                backgroundSize: '200px',
                backgroundRepeat: 'repeat',
              }}
            />
            <div className="absolute -inset-1 rounded-2xl ring-1 ring-[hsl(var(--accent))]/40 pointer-events-none" />
            <img
              src="/logo.png"
              alt="Markaz Fiqih"
              className="relative w-full h-full object-contain p-4 brightness-0 invert"
            />
          </div>

          <h2 className="font-serif text-2xl font-bold text-foreground">
            Yuk, Masuk Dulu
          </h2>
          <div className="w-10 h-[3px] rounded-full bg-[hsl(var(--accent))] mx-auto mt-3 mb-4" />
          <p className="text-muted-foreground mb-8">
            Masuk sebentar untuk lanjut menjelajahi kelas-kelas fiqih pilihanmu.
          </p>

          <Button asChild className="w-full h-12 text-base shadow-md shadow-primary/20" size="lg">
            <Link href={`/login?redirect=${encodeURIComponent(location)}`}>
              <LogIn className="h-4 w-4 mr-2" />
              Masuk dengan Google
            </Link>
          </Button>

          <p className="text-xs text-muted-foreground/70 mt-4">
            Belum pernah ke sini? Akun baru otomatis dibuat saat pertama kali masuk.
          </p>
        </div>
      </div>
    );
  }

  return <>{children}</>;
}
```

## Penjelasan Perubahan Desain
1. **Logo asli, bukan ikon generik** — badge sekarang menampilkan
   `/logo.png` (sudah ada di public folder) di atas background gradient
   merah, bukan ikon `BookLock` dari lucide-react.
2. **Tekstur `hero-pattern.png`** ditambahkan dua lapis: sangat samar di
   background penuh halaman (opacity 4%), dan lebih terlihat di dalam
   badge logo (opacity 20%, blend overlay) — pola ini sama persis dengan
   yang dipakai di sisi visual `LoginPage.tsx`, supaya kedua halaman
   terasa satu keluarga desain.
3. **Ring gold tipis** di sekeliling badge logo, aksen kecil yang
   menegaskan warna gold brand tanpa berlebihan.
4. **Garis aksen gold pendek** di bawah judul "Yuk, Masuk Dulu" sebagai
   pemisah visual antara judul dan deskripsi, pola divider kecil yang
   konsisten dengan section lain di landing page.
5. **Shadow lebih hidup** — card dan tombol dapat shadow tint warna
   primary/gold (bukan shadow abu-abu generik) supaya berasa lebih
   "branded".
6. Icon `LogIn` di tombol tetap dipertahankan (sudah pas), hanya tombol
   dibuat sedikit lebih tinggi (`h-12`) senada dengan tombol di
   `LoginPage.tsx`.

## Yang Tidak Boleh Berubah
- Logic auth (`useAuth`, redirect URL, loading state) TIDAK diubah sama
  sekali.
- Halaman `LoginPage.tsx` tidak disentuh (hanya dijadikan referensi
  desain).
- Tidak ada perubahan database/SQL untuk prompt ini.
