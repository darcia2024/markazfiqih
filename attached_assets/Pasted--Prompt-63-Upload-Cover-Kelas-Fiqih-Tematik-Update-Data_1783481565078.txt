## Langkah 3 — Verifikasi

Setelah SQL di atas saya jalankan, jalankan query berikut untuk
memastikan semua ter-update dengan benar:
```sql
SELECT title, cover_image FROM classes WHERE category = 'Fiqih Tematik' ORDER BY title;
```
Pastikan hanya "Fiqih Safar" yang `cover_image`-nya masih kosong/lama
(itu memang belum ada covernya) — 20 kelas lain harus sudah menunjuk ke
path `/covers/...` yang sesuai.

## Yang Tidak Boleh Berubah
- Tidak ada perubahan kode di halaman manapun — `cover_image` sudah
  dibaca dari database di semua tempat yang menampilkan kelas (Katalog,
  landing page, detail kelas), jadi begitu data ini terisi, tampilan akan
  otomatis update tanpa perlu ubah kode.
- File cover tidak dikompres/diubah — gunakan file asli yang sudah
  disediakan.

## Setelah Selesai
1. Konfirmasi ke-20 file sudah berada di `public/covers/`
2. Tampilkan ulang SQL lengkap di atas untuk saya jalankan manual
3. Setelah saya konfirmasi sudah menjalankan SQL, tidak perlu tindakan
   lanjutan — cukup ingatkan saya untuk cek tampilan Katalog/landing page
   guna memastikan cover sudah tampil dengan benar.
