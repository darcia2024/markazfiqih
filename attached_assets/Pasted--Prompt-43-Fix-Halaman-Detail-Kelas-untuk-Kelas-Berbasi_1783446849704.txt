# Prompt 43: Fix Halaman Detail Kelas untuk Kelas Berbasis Playlist (Stats, Cover, Sidebar)

## Konteks
`ClassDetailPage.tsx` sudah punya penanganan yang benar untuk bagian
"Kurikulum" (menampilkan pesan khusus untuk kelas playlist, bukan
accordion modul kosong). Tapi ada 3 bagian lain di halaman yang sama BELUM
menangani kondisi kelas playlist (`cls.youtubePlaylistId && cls.modules
.length === 0`), sehingga masih menampilkan "0 modul" dan "Total 0 menit"
mentah-mentah:

1. **Quick stats row** (baris ~208-222, di bawah judul kelas)
2. **Sidebar "Yang kamu dapatkan"** (baris ~469-489)
3. **Cover image kelas** yang tampil abu-abu polos karena `cls.coverImage`
   kosong di database untuk kelas-kelas Fiqih Tematik (belum diisi admin)

Field `meetingCount` sudah tersedia di data kelas (ditambahkan pada
perbaikan sebelumnya, sudah ada di `getClassById` hasil `src/lib/db.ts`).

## Fix

Edit `artifacts/markaz-fiqh/src/pages/ClassDetailPage.tsx`.

```tsx
<div className="flex flex-wrap gap-4 text-sm text-muted-foreground pt-1">
  <span className="flex items-center gap-1.5">
    <BookOpen className="w-4 h-4" />

  </span>
  <span className="flex items-center gap-1.5">
```

Menjadi kondisional berdasarkan apakah kelas playlist atau modul biasa:

```tsx
  {cls.youtubePlaylistId && cls.modules.length === 0 ? (

```
(`PlaySquare` sudah di-import di file ini, dipakai di bagian Kurikulum.)

### 3. Cover Image Fallback
Cover image sekarang menampilkan `<img src={cls.coverImage} />` polos —
kalau `cls.coverImage` kosong/string kosong, browser menampilkan gambar
rusak atau area abu-abu kosong (sesuai screenshot). Tambahkan fallback:

```tsx
<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  transition={{ duration: 0.5, delay: 0.1 }}
  className="aspect-video rounded-xl overflow-hidden bg-muted shadow-sm"
>
  {cls.coverImage ? (
    <img
      src={cls.coverImage}
      alt={cls.title}
      className="w-full h-full object-cover"
    />
  ) : (
    <div className="w-full h-full flex flex-col items-center justify-center gap-2 text-muted-foreground">
      <PlaySquare className="w-10 h-10 opacity-40" />
      <span className="text-sm font-medium opacity-60">{cls.title}</span>
    </div>
  )}
</motion.div>
```
(Fallback ini sekadar placeholder yang lebih rapi daripada gambar rusak —
solusi jangka panjang untuk cover image asli adalah admin mengupload
gambar cover lewat Admin Panel per kelas, itu di luar cakupan perbaikan
kode ini.)
