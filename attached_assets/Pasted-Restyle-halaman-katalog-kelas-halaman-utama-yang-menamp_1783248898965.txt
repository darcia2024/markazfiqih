Restyle halaman katalog kelas (halaman utama yang menampilkan daftar kelas) mengikuti struktur layout berikut, sambil tetap menggunakan design system yang sudah ada di file markazfiqih-DESIGN.md sebagai acuan warna dan tipografi — jangan pakai warna biru/gelap, semua warna harus merujuk ke palet merah (#A31F2C), krem (#FBF7F4), dan emas (#B8862E) yang sudah didefinisikan di file itu.
Struktur halaman terdiri dari sidebar kiri fixed dengan lebar sekitar 240px, berisi logo Markaz Fiqih di paling atas, lalu menu navigasi vertikal: "Katalog Kelas" (aktif secara default, ikon grid/rumah, dengan indicator active state border-left 3px warna merah #A31F2C sesuai spesifikasi Header Navigation Item di design system), "Kelas Saya" (ikon buku), dan jika user berperan admin, tambahkan menu "Panel Admin" (ikon gear) di bagian bawah sidebar terpisah dengan divider. Background sidebar putih dengan border kanan 1px warna #E8DEDA.
Di area konten utama sebelah kanan sidebar, buat header halaman berisi judul "Jelajahi Kelas" menggunakan font Plus Jakarta Sans ukuran 32px weight 700 warna #241C1B (sesuai Heading 1 di design system), dengan ikon notifikasi bell dan avatar user di pojok kanan atas header.
Di bawah judul, buat search bar full-width dengan placeholder "Cari kelas fiqih..." menggunakan style Text Input dari design system (border-radius 8px, border 1px solid #E8DEDA, height 44px, focus state border merah dengan box-shadow tint merah pucat #FBEEEF). Sejajar dengan search bar di sebelah kanan, tambahkan tiga filter chip berbentuk pill/rounded dengan border-radius 999px: "Level" (dropdown: Semua, Pemula, Menengah, Lanjutan), "Kategori" (dropdown: Semua, Fiqih Tematik, Fiqih Kitab, Akademi), dan "Urutkan" (dropdown: Terbaru, Termurah, Termahal, Terpopuler). Style chip menggunakan Secondary Button dari design system tapi ukuran lebih kecil (padding 8px 16px, font-size 13px).
Di bawah search bar, buat section "Instruktur" dengan judul menggunakan Heading 3 (20px weight 600), berisi horizontal scroll atau grid berisi card kecil tiap instruktur dari tabel instructors: avatar bulat (border-radius 999px, 48px), nama instruktur (14px weight 600 warna #241C1B), dan jumlah kelas yang diajar dihitung dari relasi ke tabel classes (contoh: "8 Kelas") dengan font 12px warna #5C4F4C. Card instruktur ini bersifat clickable untuk filter katalog berdasarkan instruktur tersebut, tapi untuk saat ini cukup buat tampilannya saja tanpa perlu logic filter aktif.
Di bawah section instruktur, buat section "Semua Kelas" dengan grid card kelas: 4 kolom di desktop (lebar lebih dari 1024px), 2 kolom di tablet, 1 kolom di mobile, dengan gap 20px sesuai Spacing System di design system. Setiap card kelas menggunakan style Kelas Card dari design system (background putih, border-radius 14px, border 1px solid #E8DEDA, box-shadow tint merah, hover state translateY(-2px)) dan berisi elemen berikut dari atas ke bawah: thumbnail gambar dengan aspect ratio 16:9 dan border-radius atas 14px, di pojok kiri atas thumbnail tampilkan badge level menggunakan style Badge dari design system dengan warna berbeda per level (Pemula: hijau sukses #DCF3E5 background dengan text #1E6B41, Menengah: emas pucat #F6EBD3 dengan text #966B1F, Lanjutan: merah pucat #FBEEEF dengan text #A31F2C) — level ini diambil dari field yang perlu ditambahkan ke tabel classes jika belum ada (misalnya kolom level dengan enum pemula/menengah/lanjutan).
Di bawah thumbnail, dalam area padding 16px, tampilkan judul kelas menggunakan Heading 4 (16px weight 600, maksimal 2 baris dengan text-overflow ellipsis jika lebih panjang), lalu di bawahnya baris kecil berisi avatar instruktur (24px bulat) dan nama instruktur dengan font 13px warna #5C4F4C. Di bawah itu, tampilkan baris metadata berisi jumlah modul (ikon buku + teks contoh "2 Modul") dan estimasi total durasi jika tersedia, dengan font 12px warna #8C7D79.
Bagian paling bawah card adalah area harga — ini bagian paling krusial, jangan sampai terlewat. Jika kelas memiliki discount_price yang tidak null, tampilkan harga base_price dengan text-decoration line-through warna #8C7D79 font-size 13px di baris atas, dan di bawahnya harga discount_price dengan warna #A31F2C font-weight 700 font-size 18px. Jika discount_price null, tampilkan hanya base_price dengan warna #241C1B font-weight 700 font-size 18px tanpa coretan apapun. Pastikan tinggi area harga konsisten antara card yang punya diskon dan yang tidak, dengan cara selalu reserve space untuk 2 baris meskipun kelas tanpa diskon hanya menampilkan 1 baris — gunakan invisible placeholder atau margin-top otomatis supaya card dalam satu grid row tetap rata tingginya (equal height).
Card yang berstatus draft tidak perlu ditampilkan di halaman katalog publik ini sama sekali — filter query untuk halaman ini hanya mengambil data dengan status published. Ambil semua data dari tabel classes, modules, dan instructors yang sudah ada di Supabase (bukan data mock/dummy lagi), gunakan query yang sudah ada atau buat baru jika belum tersedia.
Terakhir, tambahkan empty state untuk kondisi ketika hasil pencarian atau filter tidak menemukan kelas apapun: tampilkan ilustrasi sederhana atau ikon besar di tengah area grid, dengan teks "Belum ada kelas yang cocok dengan pencarianmu" menggunakan font 16px warna #5C4F4C, dan tombol kecil "Reset Filter" menggunakan Ghost Button dari design system di bawahnya.
