# Prompt 134: FIX BUG — Klik Bintang Kosong Gak Kedaftar (StarRating)

## Konteks — Bug Dikonfirmasi
`components/StarRating.tsx` render tiap bintang sebagai `<svg fill={filled
? 'currentColor' : 'none'}>`. Saat bintang belum ke-filled (`fill="none"`),
perilaku default browser membuat AREA DALAM bintang itu TIDAK MERESPON
klik sama sekali — cuma garis stroke tipis di pinggirnya yang bisa
kedeteksi klik. Ini bikin form review (dan fitur rating pengajar,
Prompt 118, yang juga pakai komponen ini) SUSAH/GAK BISA dipakai — user
klik di tengah bintang (area yang keliatan kosong), klik-nya tidak
terdaftar, `onChange` tidak pernah terpanggil, rating tetap 0.

## Fix — Tambahkan `pointerEvents: 'all'` ke Tiap SVG Bintang

Edit `components/StarRating.tsx`, cari elemen `<svg>` di dalam
`.map()`, tambahkan `style={{ pointerEvents: 'all' }}` supaya seluruh
area bounding box bintang (bukan cuma garis stroke) bisa merespon
klik/hover, terlepas dari nilai `fill`:

```tsx
{[1, 2, 3, 4, 5].map((star) => {
  const filled = star <= display;
  return (
    <svg
      key={star}
      viewBox="0 0 20 20"
      fill={filled ? 'currentColor' : 'none'}
      stroke="currentColor"
      strokeWidth={1.5}
      style={{ pointerEvents: 'all' }}
      className={cn(
        SIZE_MAP[size],
        filled ? 'text-yellow-400' : 'text-muted-foreground/40',
        interactive && 'cursor-pointer transition-colors duration-100',
        interactive && star <= hovered && 'text-yellow-400',
      )}
      onMouseEnter={() => interactive && setHovered(star)}
      onClick={() => interactive && onChange?.(star)}
    >
      <path
        strokeLinecap="round"
        strokeLinejoin="round"
        d="M9.049 2.927c.3-.921 1.603-.921 1.902 0l1.07 3.292a1 1 0 00.95.69h3.462c.969 0 1.371 1.24.588 1.81l-2.8 2.034a1 1 0 00-.364 1.118l1.07 3.292c.3.921-.755 1.688-1.54 1.118l-2.8-2.034a1 1 0 00-1.175 0l-2.8 2.034c-.784.57-1.838-.197-1.539-1.118l1.07-3.292a1 1 0 00-.364-1.118L2.98 8.72c-.783-.57-.38-1.81.588-1.81h3.461a1 1 0 00.951-.69l1.07-3.292z"
      />
    </svg>
  );
})}
```

(Perubahan HANYA menambahkan satu baris `style={{ pointerEvents: 'all'
}}` — sisanya PERSIS sama seperti kode yang sudah ada, JANGAN ubah
bagian lain dari komponen ini.)
